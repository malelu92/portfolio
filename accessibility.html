Accessibility Features

This project consisted of developing different methods for making webpages more accessible to any type of users, including visually impaired people on different degrees and people with motor disabilities. Each of these projects is described below and can be used in this portfolio, following the instructions available for each one. The features should be used on laptops with Google Chrome.

Screen Reader

Highly visually impaired users are unable to read the text available on websites. They have to sue the keyboard to navigate and can overcome this issue by activating the text-to-speech function available on computers. In this project, I recreated this functionality, making it available for users to hear the text by pressing the keys:

Shift + down arrow to read the text from top to bottom on each click, indicating which of them are links

Shift + up arrow to read the text from bottom to top on each click, indicating which of them are links

Tab to go through all the links from top to bottom

Shift + Tab to go through all the links from bottom to top




Magnification

Users with a certain degree of visual impairment, who can read with a certain effort, may need to use bigger fonts in order to do it more comfortable. The screen's automatic zoom is not the ideal solution, because it forces the user to scroll the screen left and right to be able to read all the text. In this project we made it possible for the user to click on the desired text and see it zoomed on the screen.

Hover mouse to select area to magnify

Space bar to add and remove zoom




Single Switch Input

Users with severe motor disabilities may only be able to use the computer through single clicks. In this project I created a single button switch, allowing the user to navigate through the page by using only one button on the keyboard.

The button activates an horizontal and a vertical bar that move on the screen and can be stopped using the same button. The point where they intersect, clicks the screen.

Buttons "up" and "down" will appear on the screen. They can be clicked in order to scroll the page.

Press b key to initiate and stop selection bars.
